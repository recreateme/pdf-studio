> **版本**：1.0.0　　**平台**：Windows / macOS / Linux  
> **定位**：WPS 免费版 PDF 工具箱补位 · PDF 核心处理本地离线 · 无广告

---

## 产品定位

PDF Studio **不是** WPS 的替代品，而是 **WPS 免费用户的 PDF 处理补位工具**：

| 场景 | 推荐使用 |
|------|----------|
| 日常写 Word/Excel/PPT、**免费 PDF 阅读与基础批注** | 继续使用 **WPS 免费版** |
| 拆分、合并、压缩、加密、水印、OCR、页面管理等 **WPS 会员 PDF 功能** | 使用 **PDF Studio**（本地免费） |

典型工作流：在 WPS 中打开 PDF 阅读 → 需要处理时拖入 PDF Studio → 处理完成后在 WPS 或本应用阅读器中查看。

首次启动会显示 **依赖向导**，列出必选/可选组件及安装命令。

## 1 快速开始

### 1.1 环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 · macOS 12+ · Ubuntu 20.04+ |
| Python | 3.10 及以上（源码运行时需要） |
| 内存 | 建议 4 GB 以上 |
| 磁盘 | 安装目录约 500 MB（含OCR模型） |

### 1.2 安装依赖（源码运行）

**推荐：使用已有 Conda 环境 `pdf-Assist`**

```bash
conda activate pdf-Assist
cd pdf_studio
pip install -r requirements.txt
playwright install chromium    # 网页转 PDF 需要
```

**或使用 venv（Python 3.10–3.12）**

```bash
cd pdf_studio
python -m venv .venv
# Windows: .venv\Scripts\activate
# Linux/macOS: source .venv/bin/activate
pip install -r requirements.txt
playwright install chromium
```

> 注意：AES-256 加密需要 `cryptography` 包（已列入 requirements.txt）。

### 1.3 启动应用

```bash
conda activate pdf-Assist   # 若使用 Conda
python main.py
```

首次启动将弹出 **欢迎向导**，展示依赖状态与推荐安装命令。勾选「下次不再显示」后可在 `~/.pdf_studio/config.json` 中重置 `setup_wizard_completed` 再次查看。

### 1.3.1 运行自动化测试

```bash
conda activate pdf-Assist
pip install -r requirements-dev.txt   # 含 pytest-qt，用于 UI smoke
pytest tests/ -v
# 或 Windows 双击：scripts\run_tests.bat
```

当前基线约 **116 passed**（未装 pytest-qt 时 1 项 UI smoke 跳过）。集成测试会使用 `D:\测试文件` 中的 PDF 与图片样本（若该目录存在）。

### 1.4 打包为可执行文件

```bash
pip install pyinstaller
pyinstaller pdf_studio.spec
# 生成文件在 dist/PDFStudio/ 目录
```

## 3 PDF 拆分

### 3.1 导入文件

- **拖拽**：将 PDF 文件拖入左侧"导入PDF"区域
- **点击选择**：点击导入区域弹出文件选择对话框

导入后，右侧面板会自动加载**页面缩略图**并显示总页数。

### 3.2 选择页面

在右侧缩略图面板中：

| 操作 | 效果 |
|------|------|
| 单击 | 选中单页 |
| Ctrl + 单击 | 追加/取消选择单页 |
| Shift + 单击 | 连续范围选择 |
| "全选"按钮 | 选中所有页面 |
| "取消选择"按钮 | 清空选择 |

### 3.3 拆分模式

| 模式 | 说明 | 示例 |
|------|------|------|
| **自定义范围** | 手动输入页码范围，或使用缩略图选择 | `1-3,5,8-10` |
| **按页数** | 每 N 页生成一个文件 | 每5页一份 |
| **按大小** | 每份不超过指定MB（近似值） | 每份≤10MB |
| **按书签** | 按PDF目录章节拆分 | 按一级书签拆分 |
| **按空白页** | 空白页作为章节分隔符 | 扫描件分章 |

**页码范围格式**（自定义范围模式）：
```
1-3        → 第1、2、3页
5          → 仅第5页
1-3,5,7-9  → 第1-3页、第5页、第7-9页
```
留空时，以缩略图中已选中的页面为准。

### 3.4 输出设置

| 项目 | 说明 |
|------|------|
| **输出目录** | 留空时自动在源文件同目录创建"文件名_拆分"子文件夹 |
| **文件名模板** | `{stem}`=原文件名，`{index:03d}`=三位序号（如001） |
| **覆盖同名文件** | 取消勾选时自动添加数字后缀避免覆盖 |

### 3.5 开始拆分

点击**"开始拆分"**，底部进度条实时显示进度。完成后自动打开输出目录。

## 4 PDF 合并

### 4.1 添加文件

- 拖拽 PDF 文件或**整个文件夹**到导入区（自动递归扫描子目录中的PDF）
- 点击"添加文件"逐一选择
- 点击"添加文件夹"选择目录

### 4.2 调整顺序

文件列表支持：
- **拖拽排序**：用鼠标拖动列表项调整顺序
- **↑ / ↓ 按钮**：上移/下移选中项
- **删除**：移除选中项
- **清空**：一键清空列表

### 4.3 重复检测

点击"检测重复"，程序用 MD5 哈希比对所有文件，发现重复时弹出提示。

### 4.4 合并选项

| 选项 | 说明 |
|------|------|
| **自动添加书签** | 用每个源文件的文件名生成 PDF 书签/目录 |
| **合并后压缩** | 输出时做一次轻度压缩优化 |
| **统一页面尺寸** | 将所有页面强制转为 A4/A3/Letter（会影响布局） |

> 合并前会自动校验列表：跳过不存在/重复的路径，并提示问题条目。

## 4.5 PDF 对比

轻量对比两个 PDF 的差异（补位 WPS 会员「PDF 对比」）：

1. 选择文件 A 与文件 B（加密 PDF 可填写打开密码）
2. 可拖放 PDF 到左侧区域，或点击「浏览」
3. 点击「开始对比」
4. 查看页数、体积、元数据、逐页文本/渲染抽样结果
5. 可选「导出报告」保存为 TXT

适合版本核对、批量质检；扫描件无文字层时会用低 DPI 渲染抽样对比。

## 5 PDF 压缩

### 5.1 压缩模式

| 模式 | 适用场景 | 压缩率 |
|------|----------|--------|
| **高质量** | 归档/打印用途，细节清晰 | 低（约20-40%） |
| **均衡模式**（推荐） | 日常使用，平衡质量与大小 | 中（约40-60%） |
| **极限压缩** | 邮件附件等，允许明显画质下降 | 高（约60-80%） |
| **智能压缩** | 保留可搜索文字层，主要压缩内嵌图 | 中（推荐扫描件） |

### 5.2 注意事项

- 压缩通过降低图像 DPI 和 JPEG 质量实现，**纯文字PDF压缩效果有限**
- 极限压缩模式会将页面重新渲染为图像，**会丢失文字层（不可搜索/复制）**
- 建议压缩前备份原文件

## 6 加密 / 解密

### 6.1 加密

1. 导入需要加密的 PDF
2. 设置**用户密码**（打开文档时需要输入）
3. 可选：设置**所有者密码**（用于修改权限，若留空与用户密码相同）
4. 选择权限：允许/禁止打印、复制文字、编辑
5. 设置输出路径，点击"加密PDF"

> 加密算法：**AES-256**（符合 PDF 规范 1.7 及以上）

### 6.2 解密

1. 导入加密的 PDF
2. 输入密码
3. 设置输出路径，点击"解密PDF"

## 9 阅读批注

轻量 PDF 阅读器，适合处理完成后在本应用内预览：

- **按需渲染**：滚动时自动加载可见页，支持大文档（无 500 页硬限制）
- 连续滚动阅读、缩放、页码跳转
- **缩放模式**：
  - **适应宽度**：按视口宽度自动计算缩放（默认，窗口或侧栏宽度变化时自动更新）
  - **适应页高**：按视口高度适配当前可见区域中心页
  - **100%**：实际大小（1:1 点阵）
  - **预设比例**（50%～400%）：固定缩放，与适应模式互斥
- 滚动浏览时页码 SpinBox 与缩略图选中项自动同步
- **拖放打开**：未打开文档时可将 PDF 拖入阅读区；已打开时也可拖入新文件替换
- **Ctrl + 滚轮**：在阅读区缩放（切换为固定缩放模式）
- **专注阅读**：工具栏按钮隐藏侧栏与工具栏，按 Esc 或右上角关闭按钮退出
- **渲染缓存**：LRU 缓存已渲染页，缩放变更时仅重渲可见范围，降低大文档内存占用
- 阅读区背景随应用深色/浅色主题切换
- **批注编辑**：Ctrl+Z / Ctrl+Y 撤销重做；切换/打开新文档时提示未保存更改
- **文字吸附**：高亮 / 下划线 / 删除线框选时自动吸附到文字边界
- **便签**：点击便签图标查看或编辑内容
- **文本框**：选中「文本框」工具后，拖拽待保存文本框右下角可调整大小
- **删除已有批注**：批注列表选中 PDF 内已有批注可标记删除，保存时生效
- **保存**：「另存为…」导出副本；「保存到原文件」直接写回原 PDF（请谨慎使用）
- **单页 / 双页**：工具栏切换并排双页视图（适应宽度按半屏计算）
- **窗口全屏**：工具栏全屏按钮或 Esc 退出（与「专注阅读」独立）
- **缩略图预览**：正文高清渲染完成前，优先复用侧边栏缩略图作为占位
- **偏好记忆**：缩放模式、固定比例、单/双页、侧栏宽度自动保存，下次打开 PDF 时恢复
- 侧边栏：缩略图（分批加载）/ 书签 / 批注列表
- 全文搜索
- 批注类型：高亮、下划线、**删除线**、文本框、**便签**、**自由绘制**、**矩形/线条**、**图章**
- 批注列表：筛选、双击跳转、删除待保存项、导出摘要 TXT
- 可保存为新 PDF

## 11 图片工具

图片工具页包含 Tab：**PDF → 图片** · **图片合并** · **图片压缩** · **图片 → PDF** · **内容提取** · **扫描增强**。

### 11.1 PDF → 图片

1. 拖入 PDF 文件，右侧显示缩略图预览
2. 可选择特定页面（留空则导出全部）
3. 设置输出格式（PNG/JPEG/WEBP/TIFF/BMP）和 DPI
4. 可开启图像增强：锐化、灰度化、去噪
5. 勾选 **「合并为长图」** 可将多页纵向拼接为一张长图
6. 设置输出目录，点击"开始转换"

**命名规则**：`原文件名_p0001.png`（页码四位补零）

### 11.2 图片 → PDF

1. 拖入图片（支持 PNG/JPG/WEBP/TIFF/BMP/GIF）
2. 或点击"添加文件夹"批量导入
3. 设置布局：
   - **单图单页**：每张图独占一页（推荐）
   - **九宫格（3×3）**：9张图拼为一页
4. 选择页面尺寸（A4/A3/Letter/原始尺寸）
5. 设置输出路径，点击"转为PDF"

### 11.3 图片合并（长图 · 网格）

将多张图片合并为 **一张或多张图片**（PNG/JPEG/WebP），适合聊天截图、证件合集、九宫格发帖。

1. 打开 **图片工具 → 图片合并** Tab
2. 拖入或选择多张图片；右侧以 **文件列表** 展示，可 **拖拽** 或 ↑↓ 调整顺序
3. **单击** 列表项在下方查看较大缩略图；**双击** 用系统默认程序打开该文件
4. 选择合并方式：
   - **纵向长图**：按顺序自上而下拼接；可勾选 **固定画布宽度**（如 1080px）统一缩放后拼接
   - **横向长图**：自左向右拼接；可勾选 **固定画布高度**
   - **网格拼图**：设置行数、列数，或使用 2×2 / 3×3 等预设；超出单页容量时自动输出多张；可自定义 **多页后缀**（默认 `_{page:03d}`，如 `合并_001.png`）
5. 可设置外边距、间距、输出格式与输出质量；WebP 格式可选 **无损**
6. 点击 **开始合并**，完成后自动打开输出目录

**与扫描增强联动**：在 **扫描增强** Tab 完成处理后，点击 **送到图片合并** 可将输出图一键导入合并列表。

> 动图 GIF 仅使用首帧。需 PDF 时可在「图片→PDF」中导入合并结果。

### 11.4 图片压缩

支持 **单张** 或 **批量** 压缩，输出到新文件（原图后缀 `_compressed`）。

1. 打开 **图片工具 → 图片压缩** Tab，拖入或选择图片（可添加文件夹）
2. 选择压缩方式：
   - **目标大小**：设定上限（KB/MB），自动调节质量与尺寸，使输出 **尽量接近且不超过** 该大小（推荐 JPEG/WebP）
   - **按比例**：按原图宽高的百分比缩放（如 80%），再按所选格式保存
   - **指定质量**：保持原图尺寸，按固定质量（10–100）有损压缩
3. 选择输出格式（自动 / JPEG / WebP / PNG）与输出目录
4. 点击 **开始压缩**

### 11.5 扫描增强

对扫描件/照片进行纠偏、去黑边、去噪、对比度/亮度、灰度化、二值化等处理，输出带 `_enhanced` 后缀的新文件。完成后可点击 **送到图片合并**，将结果一键导入合并列表。

### 11.6 内容提取

从 PDF 中提取纯文字或内嵌图片，无需 WPS 会员。

| 参数 | 说明 |
|------|------|
| **渲染DPI** | 越高识别越准但速度越慢，建议 150-200 |
| **置信度阈值** | 低于此值的识别结果将被丢弃，建议 40-60% |

### 12.3 导出格式

| 格式 | 说明 |
|------|------|
| **TXT** | 纯文本，每页以分隔线区分 |
| **DOCX** | Word 文档，保留段落结构 |
| **Markdown** | Markdown 格式，便于进一步处理 |
| **JSON** | 结构化数据，含坐标和置信度信息 |
| **可搜索PDF** | 在原PDF上叠加透明文字层，保留原始外观且可搜索/复制 |

### 12.4 首次使用

RapidOCR 引擎模型文件会在**首次识别时自动下载**（约 20-50 MB）。  
完成后完全离线运行，无需网络。

## 13 网页转 PDF / 截图

### 13.1 转 PDF

1. 在"URL输入"框填入网址（如 `https://example.com`）
2. 设置输出目录
3. 点击"开始转换"

### 13.2 批量转换

在"批量模式"文本框中每行输入一个URL，或点击"从TXT/CSV导入URL"批量导入。多个 URL 将按设置页 **批量并行数**（默认 2）同时转换，加快速度。

### 13.3 页面加载设置

| 参数 | 说明 |
|------|------|
| **超时时间** | 页面加载超时秒数（默认30s） |
| **加载后等待** | JS渲染等待时间（动态内容需要） |
| **滚动到底部** | 触发懒加载图片和内容（推荐开启） |
| **最大滚动次数** | 滚动次数上限，防止无限滚动页面卡死 |
| **阅读模式** | 自动移除广告/固定导航栏/弹窗等干扰 |

### 13.4 登录态支持

对于需要登录的页面：
1. 在浏览器中登录目标网站
2. 使用浏览器扩展（如 Cookie Editor）导出 Cookies 为 JSON 文件
3. 在"Cookie JSON 文件路径"中选择该文件

### 13.5 网页长截图

单 URL 模式下，点击 **「网页截图」** 可导出 PNG/JPEG 长图（需 Playwright 或系统浏览器）。

### 13.6 注意事项

- 首次使用需要 Playwright 已安装并下载 Chromium（约 150 MB）
- 某些网站禁止爬取，转换可能失败或内容不完整
- 转换登录页面时注意 Cookie 安全

## 14 批处理工作流中心

批处理中心可将多个 PDF 文件批量经过一系列处理步骤后统一输出。

### 14.1 工作流步骤

可自由勾选以下步骤（按顺序执行）：

```
导入文件夹/文件
      ↓
  ① OCR 识别（可选）
      ↓
  ② PDF 压缩（可选）
      ↓
  ③ 添加水印（可选）
      ↓
  ④ 添加页码（可选）
      ↓
  ⑤ PDF 加密（可选）
      ↓
  ⑥ 导出 PNG（可选）
      ↓
  输出归档到指定目录
```

### 14.2 使用步骤

1. 点击"添加文件"或"添加文件夹"导入 PDF
2. 勾选需要执行的工作流步骤并配置参数
3. 设置输出目录
4. 点击"启动批处理"
5. 右侧日志面板实时查看处理进度
6. **历史记录** Tab：查看过往运行摘要，双击或点「应用配置」可恢复步骤勾选（不含文件列表与加密密码；需在设置中开启「保存工作流历史」）

### 14.3 注意

- 任务执行中可点击"取消"中止剩余任务
- 已处理的文件不会撤销
- 各步骤失败不影响其他步骤继续执行

## 16 常见问题

发布前验收请参考 [发布冒烟清单.md](发布冒烟清单.md)。

### Q：和 WPS 是什么关系？
**A**：PDF Studio 补位 WPS **免费版**无法使用的 PDF 会员功能（拆分、合并、压缩等），建议在 WPS 中阅读，在本工具中处理。

### Q：导入 PDF 后缩略图不显示？
**A**：PyMuPDF 未正确安装。运行 `pip install PyMuPDF` 并重启应用。

### Q：OCR 识别结果乱码？
**A**：请确认已勾选正确的识别语言。中文文档需勾选"中文"，日文文档需勾选"日文"。

### Q：网页转PDF时提示"Playwright未安装"或浏览器无法启动？
**A**：
1. 确保已安装 `playwright`：`pip install playwright`
2. **优先**：若本机已安装 Google Chrome 或 Microsoft Edge，程序会自动调用系统浏览器，无需额外下载
3. 若无系统浏览器，再运行：`playwright install chromium`（需可访问官方 CDN；国内镜像若 404 可改用系统 Chrome）

### Q：压缩后文件比原来更大？
**A**：纯文字PDF体积极小，压缩重新渲染会引入图像编码开销，导致体积增大。压缩功能主要对含有大量图片的PDF有效。

### Q：批处理中某个文件失败，其他文件是否继续？
**A**：是的。批处理采用独立错误处理，单个文件失败不影响其他文件的处理。失败详情会显示在日志面板。

### Q：如何处理加密的PDF？
**A**：在"加密/解密"页面选择"解密"标签，输入密码后导出无密码版本，再进行其他处理。

### Q：应用数据存储在哪里？
**A**：
- **配置文件**：`~/.pdf_studio/config.json`
- **日志文件**：`./logs/`（应用目录下）
- **临时文件**：`./cache/temp/`（可在设置中一键清理）

### Q：最近文件列表如何清除？
**A**：在首页点击"清除记录"，或在设置页面中调低最大最近文件数后保存。
